AI Attendance

About

AI Attendance is an Android application that helps users manage attendance, track holidays, calculate attendance percentage, and securely store attendance records using Firebase Authentication and Cloud Firestore.

Features

✔ Secure Google Sign-In
✔ Attendance Tracking
✔ Holiday Management
✔ Attendance Analytics
✔ Firebase Cloud Database
✔ Android Java Application
